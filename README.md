# 📊 Seaborn Tutorial

A comprehensive Jupyter Notebook guide to **Seaborn** - a powerful Python library for statistical data visualization built on top of Matplotlib.

---

## 📋 Overview

This notebook provides a complete introduction to **Seaborn**, covering its key features, plot types, and best practices for creating beautiful and informative statistical visualizations.

### ✨ Topics Covered

- **Introduction & Advantages** of Seaborn
- Installation and Import
- Core Plot Categories:
  - **Distribution Plots**: `histplot()`, `jointplot()`, `pairplot()`
  - **Categorical Plots**: `barplot()`, `countplot()`, `boxplot()`, `violinplot()`
  - **Matrix Plots**: `heatmap()`
  - **Regression Plots**: `regplot()`, `lmplot()`
- Univariate, Bivariate, and Multivariate Analysis
- Real-world dataset examples using sales data
- Customization with palettes, styles, and statistical annotations
